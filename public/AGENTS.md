# public

## Purpose
Next.js 정적 에셋 디렉토리. 빌드 결과물과 무관하게 `/` 경로로 직접 서빙되는 파일을 포함한다. 현재는 SVG 아이콘 파일만 존재하며, 앱 UI에서는 사용되지 않는다 (create-next-app 기본 생성 파일).

## Key Files

| File | Description |
|------|-------------|
| `file.svg` | 파일 아이콘 SVG |
| `globe.svg` | 지구본 아이콘 SVG |
| `next.svg` | Next.js 로고 SVG |
| `vercel.svg` | Vercel 로고 SVG |
| `window.svg` | 윈도우 아이콘 SVG |

## For AI Agents

### Working In This Directory
- 앱 favicon은 `src/app/favicon.ico`에 위치 (여기가 아님)
- 이 폴더의 파일은 현재 앱에서 실제로 사용되지 않음 (create-next-app 기본값)
- 새 정적 파일(폰트, 이미지 등) 추가 시 이 디렉토리에 배치
